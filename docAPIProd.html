Documentação da API REST - Veículos

1. Listar Todos os Veículos

Método: GET

Rota: /veiculos

Exemplo de resposta:
[ 
  { 
    "id": 1, 
    "marca": "Ford", 
    "modelo": "Fiesta", 
    "ano": 2015, 
    "preco": 30000.00 
  } 
]

2. Listar um Veículo Específico

Método: GET

Rota: /veiculos/<int:veiculo_id>

Exemplo de resposta:
{ 
  "id": 1, 
  "marca": "Ford", 
  "modelo": "Fiesta", 
  "ano": 2015, 
  "preco": 30000.00 
}

3. Criar um Novo Veículo

Método: POST

Rota: /veiculos

Corpo da requisição (JSON):
{
  "marca": "Toyota",
  "modelo": "Corolla",
  "ano": 2020,
  "preco": 85000.00
}

Resposta:
{
  "id": 2,
  "marca": "Toyota",
  "modelo": "Corolla",
  "ano": 2020,
  "preco": 85000.00
}

4. Atualizar um Veículo

Método: PUT

Rota: /veiculos/<int:veiculo_id>

Corpo da requisição (JSON):
{
  "marca": "Honda",
  "modelo": "Civic",
  "ano": 2019,
  "preco": 90000.00
}

Resposta:
{
  "id": 1,
  "marca": "Honda",
  "modelo": "Civic",
  "ano": 2019,
  "preco": 90000.00
}

5. Deletar um Veículo

Método: DELETE

Rota: /veiculos/<int:veiculo_id>

Resposta:
{
  "mensagem": "Veículo excluído com sucesso"
}

6. Criar Veículos e Listar com index.html

Método: GET e POST (via index.html)

Interação: Pelo formulário de cadastro de veículos na página index.html

Para **listar os veículos** ou **adicionar um novo** via interface, use a página index.html.

- **Listar todos os veículos**: A página carrega a lista automaticamente quando você a acessa.
- **Cadastrar novo veículo**: Preencha o formulário e clique em "Adicionar Veículo" para enviar via API (POST). A lista será atualizada imediatamente com o novo veículo.

Exemplo de como adicionar um novo veículo:
{
  "marca": "Chevrolet",
  "modelo": "Onix",
  "ano": 2021,
  "preco": 40000.00
}
